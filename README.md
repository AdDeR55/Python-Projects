# Python-Projects

1. 🌤 Live Weather Dashboard

## 📌 Overview
Live Weather Dashboard is an interactive web application that allows users to search for any city worldwide and view real-time weather conditions on a map. The app includes a secure login system and lets users set alerts for temperature spikes at specific locations. User credentials are securely stored in a separate JSON file.

## 🚀 Features
- 🌍 **Search Any City**: View weather details for any location across the globe.
- 🗺 **Map Integration**: Display searched cities on an interactive map.
- 🔐 **Login System**: Secure authentication using a username and password.
- ⚠ **Temperature Alerts**: Set notifications for temperature rises at selected locations.
